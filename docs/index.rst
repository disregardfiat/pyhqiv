pyhqiv — Horizon-Quantized Informational Vacuum (HQIV)
======================================================

**pyhqiv** implements the Horizon-Quantized Informational Vacuum (HQIV) framework
from the paper: *Ettinger, Steven Jr*, *Horizon-Quantized Informational Vacuum (HQIV): A Unified Framework from Causal Horizon Monogamy and Discrete Null-Lattice Combinatorics*. Zenodo, 2026. `DOI: 10.5281/zenodo.18794889 <https://doi.org/10.5281/zenodo.18794889>`_

Installation
------------

.. code-block:: bash

   pip install pyhqiv

Optional extras: ``pip install pyhqiv[ase,mda,qutip,jax,pyvista]`` or ``pyhqiv[all]``.
